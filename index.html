Foodut

This web application is designed with a marketplace theme. The front-end is built using Vue.js, making use of Vuexy templates. For the backend, the Go programming language is utilized, with MySQL via XAMPP serving as the database system. The design pattern employed includes Singleton for the database connection and a Factory-like approach for handling JSON responses.

Grimoire

Grimoire is an Android notes app developed using Kotlin and SQLite. It offers a comprehensive solution for note-taking, allowing users to store both text and image content. In addition, Grimoire includes a convenient to-do list feature with a user-friendly calendar UI.

LIGHT

Tic Tac Toe AI

This tic-tac-toe game incorporates artificial intelligence, featuring three distinct agents: the Random Agent, Minimax Agent, and Alpha-Beta Pruning Agent. Players have the option to choose from board sizes of 3x3, 5x5, and 7x7. It's important to note that larger board sizes with intelligent agents may result in extended loading times, making them less playable.

HHO-CVRP

HHO-CVRP is an algorithm developed for my bachelor's degree final assignment that revolutionizes vehicle routing. It offers an efficient solution to optimize routes and minimize costs in scenarios with capacity constraints. Inspired by the hunting behavior of Harris hawks, HHO-CVRP incorporates intelligent search and exploration capabilities, introducing a unique approach to the field of vehicle routing.
